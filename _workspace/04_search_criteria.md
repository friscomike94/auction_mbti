# 04. 경매 검색 기준 매핑 (진단 결과 → 법원경매 검색조건)

> **목적**: 경매 진단 결과(유형코드 + profile_inputs)를 `courtauction.go.kr`에서 바로 거를 수 있는 구체적 검색조건으로 **결정적**으로 변환한다. 같은 입력이면 항상 같은 조건.
> **핵심 공식**: `검색조건 = f(유형 축, profile 입력값)`. **입력값(지역·자금·규제·보유)이 뼈대, 유형 축이 취향(물건종류·유찰횟수·권리난이도).**
> **정본**: 유형코드·축·profile_inputs = `02_gyeongmae.json` 그대로. 규제 수치 = `00_regulations.md`(2026-07-01)만 인용.
> 짝 파일: `04_search_criteria.json`(기계가 읽는 규칙 + 골든 케이스). 웹은 JSON 규칙을 JS로 재현, QA는 골든으로 검증.

> ⚠️ **면책**: 본 매핑은 투자성향 기반 물건 탐색 가이드일 뿐 **투자·세무·법률 자문이 아니다.** 부동산 정책은 수시 변경된다. 실제 입찰·대출·세금 적용 전 **금융기관·세무사·법무사**와 해당 시점 규정을 직접 확인하라.

---

## 0. 입력의 두 갈래

| 갈래 | 출처 | 결정하는 검색 차원 |
|---|---|---|
| **사실(뼈대)** — 사용자가 고른 값 | profile_inputs: `region_pref`, `fund_size`, `ownership`, `region_regulated`, `purpose`, `experience` | region, price, regulated_preference |
| **취향(성향)** — 유형 축 | 코드 4자리: FUND(s/b)·DIFFICULTY(c/x)·REGION(n/w)·PURPOSE(r/f) | property_type, min_fail_count, rights_difficulty, area |

예) "경기·비규제·5천·무주택" = 사실 → region/price/preference. "A-scnr(소액·안전·연고·실수요)" = 취향 → 아파트·유찰1회·권리단순.

## 1. 차원별 도출 로직

### 1-1. region (시/도 + 시군구) — **사이트 필터 O**
`region_pref` 입력에서 직접:
- `home`(연고) → 연고 시/군/구 1~2곳 (REGION=n과 일치)
- `metro` → 수도권(서울·경기·인천)
- `nation` → 전국, 수익률 상위 복수 시군구 순회 (REGION=w)

REGION 축이 미세조정: `n`→시군구로 좁힘 / `w`→광역·복수로 넓힘.
※ 시군구 단위 입력은 profile_inputs에 없음 → '연고 시군구'는 결과화면에서 사용자가 직접 지정(diagnostic-designer 협의 대상).

### 1-2. regulated_preference (비규제선호/무관/규제도가능) — **사이트 필터 X → region 조정 + notes**
courtauction에 규제지역 필터가 없으므로 region 선택을 조정하고 notes로 안내한다.

| 보유×용도 | 선호 | 근거(00_reg) |
|---|---|---|
| 다주택(multi) | **비규제선호(강)** | 취득세 8~12% 중과 회피 + 규제지역 경락대출 LTV 0% 위험 (B-2, A-1, D) |
| 1주택(one) | 비규제권장 | 규제지역 추가구입 주담대 금지 위험 (A-1) |
| 무주택+실거주(none·reside) | **규제무관(규제지역도 적합·기회)** | 경매는 토허 2년 실거주의무·자금조달계획서 면제 → 규제지역을 의무 없이 취득(기회). 단 LTV40%·6개월 전입·고가낙찰 유의 (D, A-2) |
| 무주택+임대(none·rent) | 규제무관~비규제선호 | |
| 단기매각(flip) | 비규제선호 | 양도세 단기중과는 동일, 규제지역 고가낙찰 쏠림 회피·회전 유리 |

입력 `region_regulated` 처리: `regulated`인데 선호와 충돌 → notes에 명시 + 02_gyeongmae.json flags 강제 재사용. `unknown` → price를 규제(보수)~비규제(확대) 범위로 양쪽 제시 + 국토부 규제지역 페이지 확인 메모.

### 1-3. property_type (물건종류) — **사이트 필터 O (용도 체크박스)**
`base_by_purpose` → `fund_modifier`:
- 용도 base: reside `[아파트]` / rent `[아파트, 오피스텔, 다세대]` / flip `[아파트, 오피스텔]`
- 소액(fund u50·u150 또는 FUND=s): `[다세대, 연립, 오피스텔, 도시형생활주택]` 추가(저가 진입)
- 대액(fund u300·o300 또는 FUND=b): `[아파트]` 우량 집중(임대면 오피스텔 허용), 다세대·연립 제외

토허구역 토지거래허가 대상 용도 = **아파트** → 일반매매면 허가 필요, 경매는 면제. 단 규제지역 아파트는 고가낙찰 쏠림 경계(notes).

### 1-4. price_min / price_max (최저매각가격) — **사이트 필터 O**, 핵심
> **레버리지·규제 대출한도를 반드시 반영한다(스킬 §3). 경락잔금대출도 규제 동일 적용(00_reg D).**

**공식**: `price_max ≈ equity_top × LTV배수 × 보수계수`
- `equity_top` = 가용자금 밴드 상단(입찰 가능한 최대 자기자본): u50=5,000만 / u150=1.5억 / u300=3억 / o300=4억
- `LTV배수 = 1/(1−LTV)`, LTV는 **00_reg 근거**:
  - 비규제 LTV 70% → **3.33** (C, B-1)
  - 규제·무주택 LTV 40% → **1.67** (생애최초 70%면 3.33로 분기) (A-2, B-1)
  - 규제·1주택(미처분 추가)·다주택 = 주담대 **금지 LTV 0%** → **1.0**(자기자본만) (A-1, B-1)
- `보수계수`(규제수치 아님, 성향·불확실성 반영): FUND `s`=0.9(레버리지 보수·다건) / `b`=1.0. 비규제 다주택은 ×0.7 추가(수도권 6억한도·추가구입 제약·DSR 불확실).

**price_min**: 소액(s)만 3,000만 하한(저가 권리·환금 불량 회피), 대액은 하한 없음(우량 집중).

**예시(스킬 §3 일치)**: 5천 + 소액분산 + 비규제 → 5,000만 × 3.33 × 0.9 ≈ **1.5억**(보수). 대출 최대·DSR 충족 시 2억 내외. 같은 5천이 규제+다주택이면 LTV 0 → **5천 내외 + 강경고**.

⚠️ **price_max는 '검색 상한'일 뿐.** 규제지역은 토허 면제로 수요 쏠려 고가낙찰 빈발 → **시세 역산 입찰 상한선**을 더 엄격히 지킬 것.

### 1-5. min_fail_count (유찰회수) — **사이트 필터 O**
`base`: 안전 c=**1** / 특수 x=**2**. modifiers:
- 단타 f: **+1**(싸게 사야 차익)
- 규제지역(region_regulated=regulated): **+1**(고가낙찰 쏠림 → 더 내려온 회차 대기)
- 대액·안전(FUND=b & DIFFICULTY=c): **−1**(우량·안전은 경쟁 심해 유찰 적음, 무리한 대기 비현실; floor 0)

1회 유찰 시 통상 최저가 20~30%↓(법원별 상이). 특수(x)는 유찰 누적이 정상.

### 1-6. rights_difficulty (안전/특수) — **사이트 필터 O (특수조건 체크박스)**
- `c` → **safe**: 말소기준 이하 전부 소멸·인수권리 0·대항력 없는 점유(인도명령 가능)·임차인 단순. 특수조건 미체크.
- `x` → **special_ok**: 유치권·법정지상권·분묘기지권·지분매각·대지권미등기·선순위(대항력)임차인 허용. 특수조건 체크.

`experience` 보정: **초보+x → safe로 강등 권고 + danger flag**(02_gyeongmae.json, 첫 5건 안전). 중급+x → 쟁점 1~2개까지. 숙련+x → 전면 허용.

### 1-7. area_pyeong (전용면적) — **사이트 필터 O**
reside → 국민평형(전용 59~84㎡), 소액이면 소형(~59㎡) / rent → 소형 임대형(~59㎡)·오피스텔 / flip → 환금 국민평형(59~84㎡), 소액이면 소형.

### 1-8. notes (사이트로 못 거르는 것) — 분리
02_gyeongmae.json flags(보유·경험·규제·축·코드 해당분) 전부 재사용 + price 분기메모(생애최초 LTV70%·규제 LTV40%·다주택 대출0·DSR40%·경락대출 금융사별 상이[불확실]) + 고가낙찰 경계 ↔ 토허 면제(기회) + 취득세/양도세/종부세 + 명도(인도명령 한정 c / 예비비 x) + 면책.

## 2. 규제를 가격대·선호에 녹인 방식 (스킬 §4)

1. **다주택 → 비규제·저가 가중 + 취득세 경고**: regulated_preference=비규제선호(강). 규제지역 선택 시 LTV 0(price_max 급감) + 취득세 8~12% 중과 notes. (G2)
2. **규제지역 경매의 양면**: (기회) 토허 2년 실거주의무·자금조달계획서 면제 → 무주택 실수요엔 규제지역도 적합 / (함정) 그 쏠림으로 고가낙찰 + 대출규제는 동일(LTV·DSR). price엔 고가낙찰 경계, regulated_preference엔 양면 명시. (G3)
3. **무주택 실수요 → 규제지역도 적합 + 전입의무**: regulated_preference=규제무관. LTV 40%(생애최초 70%)·6개월 전입의무 price 분기 + notes. (G1, G3)
4. **자금→가격대 환산에 대출한도 직접 반영**: 비규제 ×3.33 / 규제무주택 ×1.67 / 대출금지 ×1.0. 보수 제시 + "대출·DSR 확인" 메모 상시.

## 3. 핵심 경매 유형(★)별 검색조건 예시

> 입력값이 비면 축 기본값 가정(자금=소액/중액, 지역=연고, 규제=비규제, 보유=무주택). 실제는 입력값으로 덮어쓴다.

### ★ A-scnr 동네 안전 낙찰러 (s·c·n·r) — 최빈
- region: 연고 시군구 / property_type: 아파트(소액이면 +오피스텔·다세대) / price_max: 자금×배수×0.9 / **min_fail≥1** / rights: **safe** / area: 소액 소형·중액 국민평형. notes: 고가낙찰 경계(초보), 무주택 우대.

### ★ A-scnf 동네 소액 단타러 (s·c·n·f)
- 위와 같되 **min_fail≥2**(+flip), area: 환금 소형, notes: 단기 양도세(다주택 중과)·매각출구 검증.

### ★ A-scwf 수익률 추적 원정러 (s·c·w·f)
- region: **전국 복수 시군구** / property_type: 환금형 아파트·오피스텔 / **min_fail≥2** / rights: safe / notes: 출구 검증 최우선·원정비용 차감·동시진행 상한.

### ★ A-bcnr 우량 실거주 낙찰러 (b·c·n·r)
- region: 연고 / property_type: **아파트(우량 집중)** / price_max: 자금×배수×1.0(대액 레버리지 적극) / **min_fail≥0**(우량 유찰 적음) / rights: safe / area: 국민평형. notes: 대액 집중=한 번 고가낙찰이 큰 손실, 시세 역산 엄수.

### ★ A-bxnf 지역 특수 수익가 (b·x·n·f)
- region: 연고 / property_type: 아파트·다세대·근린 등 / **min_fail≥3**(특수2+flip1) / rights: **special_ok**(유치권·지분 등) / notes: **숙련 전제**·최악 시나리오 입찰가·소송/명도 예비비·다주택 단기 양도세.

### ★ A-bxwf 전업 특수물건 전문가 (b·x·w·f)
- region: 전국 / property_type: 전 용도 / **min_fail≥3** / rights: **special_ok(최고난도)** / notes: 전업/숙련 전용·건별 최악 시나리오 자금 별도 적립·법인/세무 설계.

## 5. 검증 체크 (QA용)
- 모든 검색 차원이 유형 축·입력값과 모순 없는가(특수선호인데 safe만? 소액인데 고가대?). → §1 규칙으로 결정적.
- 사이트에서 거를 수 있는가: region·property_type·price·min_fail·rights(특수조건)·area = **필터 O**. regulated_preference·세금·대출가용액·명도 = **notes**(필터 X).
- 규제 수치가 00_regulations.md에만 근거하는가(LTV 70/40/0, 취득세 8~12%, 양도세 중과, 토허 면제). 면책 유지.
