# Real Estate Strategy System

부동산 투자성향 테스트와 이후 강의자료, 리포트, 랜딩페이지에 공통 적용할 디자인 시스템입니다.

## 1. 디자인 방향

- 차분한 투자 브랜드 톤: 과한 네온보다 신뢰감, 판단력, 자산관리 느낌을 우선합니다.
- 프리미엄 에디토리얼: 넓은 여백, 굵은 제목, 얇은 라인, 낮은 채도의 컬러를 사용합니다.
- 자료 재사용성: 웹, 강의 슬라이드, PDF, 체크리스트, 진단 리포트에 같은 색과 카드 규칙을 씁니다.
- 정보 우선: 장식은 배경과 구분선 정도로 제한하고, 문항과 결과 해석의 가독성을 우선합니다.

## 3. 타이포그래피

- 기본 폰트: Noto Sans KR
- 제목: 800-900 weight, letter spacing -5% 내외
- 본문: 500-650 weight, line height 1.65-1.8
- 라벨: 800-900 weight, 작은 크기, uppercase 또는 짧은 한글 라벨

권장 크기:

| 역할 | Web | Slide |
| --- | --- | --- |
| Hero title | 64-84px | 48-72pt |
| Section title | 24-32px | 28-40pt |
| Body | 16-18px | 18-24pt |
| Label | 11-13px | 12-16pt |

## 4. 레이아웃 원칙

- 최대 폭: 웹 기준 1040px
- 카드 radius: 22-36px
- 카드 내부 padding: 22-48px
- 주요 섹션 간격: 28-48px
- 카드 배경은 흰색에 가까운 아이보리, 그림자는 낮게 사용합니다.
- 배경 장식은 10-14% opacity 이하의 큰 형태만 사용합니다.

## 5. 컴포넌트 규칙

### Hero

- 큰 제목 2줄 구조를 기본으로 합니다.
- 강조 단어는 Navy에서 Sage로 이어지는 낮은 채도 그라디언트를 씁니다.
- 배경에는 라인 그리드와 흐릿한 대형 도형만 사용합니다.

### Card

- Surface 배경 + 얇은 라인 + 약한 그림자.
- Hover는 3-5px 위로 이동하는 정도만 허용합니다.
- 선택 상태는 Navy 라인과 작은 선택 배지로 표시합니다.

### Button

- Primary: Navy 배경, 아이보리 텍스트.
- Ghost: Surface 배경, Navy hover.
- Disabled: 회색 처리, 그림자 제거.

### Result

- 결과 코드는 Navy 단색으로 강하게 보여줍니다.
- 축 카드에는 Sand Soft를 사용해 경계형을 표시합니다.
- Identity 패널은 Navy 그라디언트로 하나만 강하게 강조합니다.

### Alert

- Info: muted blue
- Warn: muted ochre
- Danger: muted brick
- 경고는 색보다 라벨과 문장으로 명확하게 전달합니다.

## 6. 앞으로 자료에 적용할 때

- 강의 슬라이드 표지는 Hero 규칙을 그대로 사용합니다.
- 본문 슬라이드는 Surface 카드 1-2개만 배치합니다.
- 체크리스트는 Card + Label + thin line 구조를 씁니다.
- 강조 색은 한 화면에 Navy, Sage, Clay 중 최대 2개만 씁니다.
- 형광색, 과한 그라디언트, 과한 블러는 사용하지 않습니다.

## 7. CSS 토큰 소스

현재 웹 테스트의 CSS 변수 영역을 기준 소스로 사용합니다. 이후 다른 자료를 만들 때는 색상과 radius, shadow, typography 규칙을 먼저 복사한 뒤 컴포넌트만 변형합니다.
